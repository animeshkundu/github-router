Or add this to .claude/settings.json:

  "env": {
    "ANTHROPIC_BASE_URL": "http://localhost:8787",
    "ANTHROPIC_API_KEY": "dummy",
    "ANTHROPIC_MODEL": "gpt-4.1"
  }
}

API

Complete API Reference

All endpoints support streaming, tool calling, and vision. OpenAI endpoints work with or without the /v1 prefix.

Endpoint Method Format Features
/v1/chat/completions POST OpenAI Chat Streaming, Vision, Tools
/v1/responses POST OpenAI Responses Codex models, Web search
/v1/messages POST Anthropic Messages Translation, Beta headers
/v1/models GET OpenAI Models Model capabilities, limits
/v1/embeddings POST OpenAI Embeddings Text embeddings
/v1/search POST Web Search Rate-limited, thread-based
/usage GET Usage Stats Quotas, reset dates
